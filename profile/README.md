# see-realview

> 이 저장소는 카카오맵을 기반으로 맛집 키워드를 검색하고 결과를 보여줍니다. 이후 맛집을 선택하면 해당 맛집에 대한 네이버 블로그 리뷰 중에서 해당 리뷰가 광고성 리뷰인지, 내돈내산 리뷰인지 판별한 결과를 미리 알 수 있도록 표시해주는 서비스 입니다. 맛집 검색 시에 활용하시면 됩니다.
